# Pokémon Emerald

This is a decompilation of Pokémon Emerald with an enhanced (~1%) shiny rate.

It builds the following ROM:

* **shiningemerald.gba** `sha1: f9d2d9d5dee7e59f7561c305303b8efe1f0b95fa`
